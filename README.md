# 육군 장교 병과 선택 가이드 🎖️

장교를 목표로 하고 계시는군요! 멋진 목표입니다. 다만 아직 성향과 적성에 대한 정보를 알려주지 않으셔서, 먼저 **주요 병과를 비교 설명**해 드리고, 이후 몇 가지 질문을 통해 맞춤 추천을 도와드릴게요.

---

## 1. 전투병과

### 🪖 보병
| 항목 | 내용 |
|------|------|
| 주요 업무 | 지상 전투의 핵심, 소대·중대 지휘, 작전 수행 |
| 필요 역량 | 강한 체력, 리더십, 결단력, 스트레스 관리 능력 |
| 장점 | 병과 규모가 가장 커서 **진급 TO가 많음**, 지휘 경험 풍부 |
| 단점 | 체력적 부담 큼, 격오지 근무 가능성 높음 |
| 적합한 유형 | 사람을 이끄는 것을 좋아하고, 야전 활동을 즐기는 활동적인 사람 |

### 🎯 포병
| 항목 | 내용 |
|------|------|
| 주요 업무 | 화력 지원, 사격 제원 계산, 화력 운용 계획 |
| 필요 역량 | 수리·계산 능력, 꼼꼼함, 공간 지각력 |
| 장점 | 전투병과 중 비교적 체계적·기술적 업무, 진급 전망 양호 |
| 단점 | 훈련 강도 높음, 정밀성에 대한 부담 |
| 적합한 유형 | 수학·물리를 좋아하고 논리적 사고가 강한 사람 |

### 🛡️ 기갑
| 항목 | 내용 |
|------|------|
| 주요 업무 | 전차·장갑차 부대 지휘, 기동전 수행 |
| 필요 역량 | 기계 이해도, 순발력, 팀워크 |
| 장점 | 강한 전우애와 소수정예 문화, 전문성 인정 |
| 단점 | 병과 규모가 작아 보직 경쟁 존재 |
| 적합한 유형 | 기계·장비에 관심 많고 팀 단위 활동을 선호하는 사람 |

## 2. 기술·전투지원병과

### 💻 정보통신
- **업무**: 네트워크·통신체계 구축 및 운용, 사이버 방호
- **역량**: IT 지식, 문제 해결 능력
- **장점**: 전역 후 IT 분야 취업 연계성 우수 ⭐
- **적합**: 컴퓨터·통신 전공자, 기술직 선호자

### 🔧 공병
- **업무**: 시설 건설, 장애물 설치/제거, 재해 복구
- **역량**: 공학적 사고, 프로젝트 관리 능력
- **장점**: 건설·토목 자격증 취득 유리, 전역 후 진로 연계 좋음
- **적합**: 건축·토목 전공자, 만들고 짓는 것을 좋아하는 사람

### 🔍 정보
- **업무**: 첩보 수집·분석, 적 동향 파악
- **역량**: 분석력, 외국어 능력, 보안 의식
- **장점**: 지적 업무 중심, 전문성 높음
- **단점**: 병과 규모 작음, 업무 성과가 드러나기 어려움
- **적합**: 분석적이고 신중하며 외국어에 강한 사람

---

## 3. 행정·군수병과

### 📊 인사행정 / 재정
- **업무**: 인사관리, 예산·회계 운영
- **장점**: 규칙적인 근무, 행정 전문성, 전역 후 공공기관 연계
- **단점**: 야전 지휘 경험 적어 고위 진급에는 불리할 수 있음
- **적합**: 꼼꼼하고 안정적인 업무 환경을 선호하는 사람

### 🚚 군수(병참·수송)
- **업무**: 보급, 물자 관리, 수송 계획
- **장점**: 물류 전문성 → 전역 후 물류·유통업계 진출 유리
- **적합**: 계획적이고 관리 능력이 뛰어난 사람

---

## 4. 진급 전망 핵심 정리 📈

- **장군 진출**: 보병 > 포병 > 기갑 순으로 TO가 많은 편 (전투병과 우세)
- **장기복무 후 전역 시 사회 연계성**: 정보통신, 공병, 군수가 유리
- **워라밸**: 행정·군수 계열이 상대적으로 양호
